# 🍔 FoodHub AI Chatbot

AI-powered customer support chatbot using LLM + SQL + Guardrails.

## 🚀 Features
- Real-time order lookup (SQL)
- AI-powered responses (OpenAI / Bedrock ready)
- Responsible AI guardrails
- Session-based memory
- Streamlit chat UI
